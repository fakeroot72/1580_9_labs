## Цель работы

Цель данной лабораторной работы заключается в освоении навыков:

* Создания программных объектов пользовательских типов с использованием классов в Python

* Перегрузки специальных методов (`__str__`, `__repr__`, `__lt__`, `__eq__`)
* Реализации конструктора, инициализации и методов класса
* Сортировки коллекций объектов пользовательских типов

   - Выводит значения полей на экран

4. **Реализуйте методы работы с коллекцией объектов**:

   - Удаление объекта из БД
   - Редактирование объекта в БД
   - Вывод БД на экран

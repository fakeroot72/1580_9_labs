# Лабораторная работа 3: Переход от структур к классам

**Алиас:** `Классы`

## Цель работы

Цель данной лабораторной работы заключается в освоении навыков:

* Создания программных объектов пользовательских типов с использованием классов в Python
* Создания модулей для описания классов
* Перегрузки специальных методов (`__str__`, `__repr__`, `__lt__`, `__eq__`)
* Реализации конструктора, инициализации и методов класса
* Сортировки коллекций объектов пользовательских типов

## Задание

### Часть 1: Базовый класс

1. **Ознакомьтесь с основными понятиями ООП в Python**:
   - Классы и объекты
   - Атрибуты и методы
   - Конструктор `__init__`
   - Специальные методы

2. **Разработайте класс "Планета"** для планет Солнечной системы. Класс должен содержать минимум 5 характеристик:
   - Название планеты
   - Радиус (км)
   - Масса (кг)
   - Расстояние от Солнца (млн км)
   - Тип (каменная, газовый гигант, ледяной гигант)

3. **Создайте модуль `planet.py`** для класса "Планета" и отладьте программу, которая:
   - Создает объекты класса "Планета"
   - Выводит значения полей на экран
   - Сохраняет данные в локальную БД (текстовый документ, более сложный вариант - JSON)

4. **Реализуйте методы работы с коллекцией объектов**:
   - Чтение из БД
   - Запись в БД
   - Сортировка БД по выбранному полю
   - Добавление нового объекта в БД
   - Удаление объекта из БД
   - Редактирование объекта в БД
   - Вывод БД на экран

### Часть 2: Специальные методы

1. **Перегрузите специальные методы**:
   - `__str__` для удобного вывода объекта
   - `__repr__` для представления объекта

2. **Создайте менеджер коллекций**:
   - Класс `PlanetCollection` для работы с коллекцией планет
   - Методы для фильтрации, поиска, статистики

### Часть 3: Сравнение и сортировка

1. **Перегрузите операции сравнения**:
   - `__lt__` для сравнения по расстоянию от Солнца
   - `__eq__` для сравнения по названию
   - `__gt__`, `__le__`, `__ge__` по необходимости

2. **Отсортируйте коллекцию планет**:
   - Используя встроенную функцию `sorted()`
   - Используя метод `sort()` списка
   - Создайте несколько вариантов сортировки (по разным полям)

### Часть 4: Работа по вариантам

**На основе разработанного класса "Планета" выполните задание по варианту:**

3. **Автомобиль**: Марка, модель, год выпуска, VIN, цвет, пробег

6. **Фильм**: Название, режиссер, год, жанр, рейтинг, продолжительность
7. **Рецепт**: Название, категория, время приготовления, ингредиенты, сложность
